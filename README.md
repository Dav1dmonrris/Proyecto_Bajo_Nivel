# Proyecto_Bajo_Nivel

Iniciar juego:
--------------
|> ¡Es necesario tener instalado las librerias de sfml! <|

* Compilar main.cpp desde terminal para crear el programa ejecutable según el sistema operativo de la computadora.

El archivo main.cpp se encuentra dentro de la carpeta JuegoPrincipal. Se recomienda que al compilar el juego desde terminal
se incluya los demás archivos .cpp de la carpeta Clases.
